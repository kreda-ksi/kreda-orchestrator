# kreda-orchestrator ξ

KREDA consists of three main parts:
- **[kreda-core-vision](https://github.com/kreda-ksi/kreda-core-vision)** (C++/OpenCV) captures footage,
- **kreda-orchestrator** (this repo, Python) dedupes, transcribes via VLM, and synthesizes notes,
- **[kreda-datasets](https://github.com/kreda-ksi/kreda-datasets)** holds test footage.
